# tickets.dev
A microservices website built using TypeScript and NATS Streaming Server

## Tech Stack:
- Angular
- TypeScript
- Express
- MongoDB
- Redis
- NATS Streaming Server
- Docker
- Kubernetes

### Requirements:
- docker
- kubernetes
- skaffold (optional)
- ingress nginx

### Getting Started
```
$ git clone https://github.com/shreyasssk/tickets.dev.git
$ cd tickets.dev
$ skaffold dev
```
